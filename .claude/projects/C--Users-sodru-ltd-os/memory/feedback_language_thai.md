description: User ต้องการให้ตอบและเขียน content เป็นภาษาไทยเสมอ รวมถึง dashboard labels และ UI text
type: feedback
---

ตอบเป็นภาษาไทยเสมอ แม้ user จะพิมพ์ภาษาอะไรก็ตาม

**Why:** User บอกชัดว่า "จำไว้ปรับให้เป็นภาษาไทย" — ต้องการภาษาไทยเป็น default ทั้ง response และ UI/content ที่สร้าง

**How to apply:**
- Response ทุก message: ภาษาไทย
- Dashboard labels, section headers, UI text: ภาษาไทย
- Comments ใน code: ภาษาไทยได้ถ้าเป็น business logic; technical terms ยังคง English
- File content (vault notes, weekly rec, insight atoms): ภาษาไทย
- Technical terms คงไว้เป็น English เสมอ ("git commit", "API key", "BUY", "SELL", ticker symbols, function names)
