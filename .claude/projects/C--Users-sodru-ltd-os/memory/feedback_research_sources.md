เวลาหาข้อมูล (stock research, paper survey, general research) ให้รวม GitHub และ Reddit ทุกครั้ง

**Why:** user ต้องการมุมมองที่กว้างกว่า mainstream — Reddit ให้ retail sentiment + bear thesis, GitHub ให้ implementation จริงและ alternative analysis

**How to apply:**
- `/stock-research` → เพิ่ม Reddit DD search (r/wallstreetbets, r/stocks, r/SecurityAnalysis) + GitHub financial model search
- `/paper-survey` → เพิ่ม GitHub implementation repo + Reddit r/algotrading practitioner discussion
- Research ทั่วไป → ถ้า search budget มี → จัดสรร 1 search slot ให้ Reddit และ 1 slot ให้ GitHub เสมอ
- ถ้า Reddit/GitHub ให้มุมขัดแย้ง mainstream → flag ⚠️ แสดงทั้งสองมุม ไม่เลือกข้างเอง
