type: user

User ให้ความสำคัญกับความคุ้มค่าสูงมาก — จะใช้ token ที่เหลืออยู่ให้หมดก่อนปิด session ไม่ปิดทิ้งเฉยๆ

**How to apply:** เมื่อ context ใกล้เต็ม อย่าแค่บอกให้ปิด — เสนองานเล็กๆ ที่ทำได้ในเวลาที่เหลือแทน เช่น commit งานค้าง, บันทึก memory, ตอบคำถามสั้นๆ
