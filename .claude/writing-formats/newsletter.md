# Newsletter format (email, Substack newsletter)

A newsletter is a recurring relationship, not a blog post. Structure serves re-opens.

## Structure
1. **Subject line** — 30-50 chars, specific, curiosity gap (not clickbait)
2. **Preview text** — 40-90 chars, extends the subject, matters for mobile
3. **Opening** — 1-2 lines, set the week's angle (not "Happy Monday")
4. **Main piece** — one idea, well-developed (400-1000 words)
5. **Quick hits** (optional) — 2-5 short notes on other things you saw
6. **Link section** (optional) — 3-7 things worth reading this week
7. **Close** — signoff with personality, not "thanks for reading"

## Length
- 500-1200 words total (short enough to read in one sitting)
- Longer is fine occasionally, but not default

## Subject line patterns
- **Specific**: "3 charts that changed my mind on rates"
- **Question**: "Is NVDA a bubble? A framework."
- **Teaser**: "The weirdest thing in Costco's 10-K"
- **Weekly stamp**: "Weekly: <specific angle this week>"

Avoid:
- "Newsletter #47"
- "My thoughts on <topic>"
- All-caps, excessive punctuation
- Anything that tips spam filters ("FREE", "$$$", "!!!")

## Voice
- More personal than blog post
- First person natural — "I noticed", "I'm watching", "I was wrong about"
- Direct address — "you" not "the reader"
- Admit uncertainty — it builds trust

## One-idea discipline
Newsletters fail when they're "here's 5 unrelated things I thought about this week." Pick ONE. The link section can have the rest.

## Thai newsletter
- Substack Thailand scene: casual + technical works well
- Email rendering: break long paragraphs (Thai is dense)
- Avoid excess English headers (confuses Thai readers scanning)

## Quick hits section (if included)
Format:
```
**Quick hits**

- <bold headline> — 1-2 sentences, link if relevant
- <bold headline> — 1-2 sentences
```

## Link section (if included)
Format:
```
**Worth your time this week**

- [Title](url) — why I liked it (1 line)
- [Title](url) — why I liked it
```
Max 5-7 links. More = no one clicks.

## Close patterns
- Tease next week if you have something planned
- Ask one question that invites reply (builds list engagement)
- Sign off with something that sounds like you

Avoid:
- "Thanks for reading!"
- "Have a great week"
- Emoji-heavy signoff

## Pre-send checklist
- [ ] Subject line tested: is it boring? rewrite
- [ ] Opening doesn't waste time on pleasantries
- [ ] Main piece has ONE point, not five
- [ ] All links work
- [ ] Quick hits aren't padding
- [ ] Close sounds like me, not a corporate memo
