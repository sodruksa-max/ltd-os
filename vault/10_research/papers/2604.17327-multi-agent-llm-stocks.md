# Multi-Agent LLM Architecture Blueprint

## สรุปสั้น
4 specialist agents วิเคราะห์คนละมุม → synthesis agent รวมด้วย NNLS embedding decomposition ผล strong-buy portfolio อยู่ที่ percentile 99.7 vs random (p=0.003) และชนะ equal-weight benchmark +25.2%

## Architecture

```
News Agent        → วิเคราะห์ข่าวล่าสุด
Fundamentals Agent → earnings, filings, valuations
Dynamics Agent    → price action, technical signals
Macro Agent       → sector + macroeconomic context
        ↓ (parallel, independent)
Synthesis Agent   → NNLS embedding decomposition → ordinal recommendation
        ↓
Strong Sell / Sell / Hold / Buy / Strong Buy
```

## Synthesis Mechanism (NNLS)
- แต่ละ agent output encode เป็น embedding
- Synthesis ทำ Non-Negative Least Squares decomposition
- คำนวณ adaptive weight ของแต่ละ agent per stock per date
- **ไม่มี agent ไหน dominate ตลอดเวลา** — weight หมุนตาม market regime

## ผล
- Strong-buy portfolio: percentile 99.7 vs random (p=0.003)
- +25.2pp excess return vs equal-weight benchmark
- Fundamentals agent dominate บน S&P 500
- Macro agent dominate บน S&P 100

## Blueprint สำหรับ Crypto Bot

| Role | Equity | Crypto Adaptation |
|------|--------|-------------------|
| Agent 1 | News | Crypto news + macro news |
| Agent 2 | Fundamentals | On-chain metrics (whale, exchange flow, active addr) |
| Agent 3 | Dynamics | VP-MACD + LOB regime + momentum score |
| Agent 4 | Macro | SPY trend + DXY + BTC dominance |
| Synthesis | NNLS | Same — adaptive weight per asset per timeframe |

## Implementation Steps

```python
# 1. ให้แต่ละ agent produce text analysis
news_analysis = news_agent.analyze(ticker, date)
onchain_analysis = onchain_agent.analyze(ticker, date)
technical_analysis = technical_agent.analyze(ticker, date)
macro_analysis = macro_agent.analyze(ticker, date)

# 2. Encode เป็น embeddings
from sentence_transformers import SentenceTransformer
model = SentenceTransformer('all-MiniLM-L6-v2')
embeddings = model.encode([news_analysis, onchain_analysis,
                           technical_analysis, macro_analysis])

# 3. NNLS synthesis
from scipy.optimize import nnls
synthesis_embedding = model.encode([synthesis_agent.synthesize(all_analyses)])
weights, _ = nnls(embeddings.T, synthesis_embedding[0])
print("Agent weights:", dict(zip(['news','onchain','technical','macro'], weights)))

# 4. Ordinal output
recommendation = synthesis_agent.recommend(all_analyses)  # 1-5
```

## Key Adaptations สำหรับ Crypto
- Rebalance รายวัน หรือรายสัปดาห์ (ไม่ใช่รายเดือนแบบ equity)
- Monte Carlo K=50,000+ (crypto volatility สูงกว่า)
- Filter universe ด้วย minimum volume ก่อน screen
- Weight Macro agent สูงขึ้นในช่วง bear market
