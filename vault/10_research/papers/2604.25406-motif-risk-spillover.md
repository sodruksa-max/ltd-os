# Motif-Based Risk Spillover Framework

## สรุปสั้น
ใช้ directed triadic motifs (3-node patterns) ใน connectedness network วิเคราะห์ว่าความเสี่ยงถ่ายเทระหว่างสินทรัพย์อย่างไร แทนที่จะดูแค่ aggregate spillover ให้ดู local pattern ที่สร้าง systemic risk — motif-based portfolio ชนะ benchmark

## Methodology
1. สร้าง quantile connectedness network (normal, median, tail conditions)
2. Extract multiscale backbone
3. หา directed triadic motifs ที่ frequency เกิน randomization baseline
4. ใส่ sector partition (colored motifs) → รู้ว่า sector ไหน drive local structure
5. กำหนด orbit position ให้แต่ละ node → รู้ว่าแต่ละ asset เป็น transmitter หรือ receiver

## Key Finding
- Assets ที่มี **orbit-position diversity สูง** ใน tail network = net spillover **transmitter**
- Transmitters = signal ว่า contagion กำลังจะแพร่ → ลด exposure
- Motif portfolio ชนะ minimum-correlation และ connectedness benchmarks (risk-adjusted)

## Cross-Asset Risk Alert สำหรับ bot

```python
import networkx as nx
import numpy as np

def compute_orbit_diversity(connectedness_matrix, assets):
    """
    ยิ่ง diversity สูง = asset นั้นเป็น transmitter ในวิกฤต
    """
    G = nx.from_numpy_array(connectedness_matrix, create_using=nx.DiGraph)
    diversity_scores = {}
    for i, asset in enumerate(assets):
        # ดู triadic patterns ที่ node นี้เป็น center
        in_deg = G.in_degree(i)
        out_deg = G.out_degree(i)
        # diversity = balance ระหว่าง roles (transmitter/receiver/mediator)
        total = in_deg + out_deg + 1e-10
        diversity_scores[asset] = 1 - abs(out_deg - in_deg) / total
    return diversity_scores

def risk_alert(diversity_scores, threshold=0.7):
    """Flag assets ที่กำลังกลายเป็น systemic transmitter"""
    alerts = [a for a, d in diversity_scores.items() if d > threshold]
    return alerts
```

## การนำไปใช้กับ crypto bot
- สร้าง network จาก correlation/return ของ BTC, ETH, altcoins, DXY, Gold
- คำนวณ diversity score รายสัปดาห์
- ถ้า BTC diversity สูงขึ้น → ลด position ป้องกัน contagion
- ใช้เป็น macro filter เพิ่มเติมจาก SPY trend check
