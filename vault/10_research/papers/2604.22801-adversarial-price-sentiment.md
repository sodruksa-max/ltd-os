# Adversarial Learning: รวม Price + Sentiment

## สรุปสั้น
ใช้ GAN (Generative Adversarial Network) รวมข้อมูล 2 ทาง: ตัวเลขราคา (quantitative) + sentiment จากข่าว (qualitative) ชนะ ARIMA และ LSTM baseline โดยเฉพาะช่วง volatility spike

## Architecture
```
Input A: OHLCV + indicators (quantitative)
Input B: Sentiment scores จาก NLP (qualitative)
         ↓
Generator  ←→  Discriminator  (adversarial training)
         ↓
Price movement prediction
```

## ผล
- เอาชนะ ARIMA และ LSTM ใน error metrics หลายตัว
- แข็งแกร่งขึ้นเป็นพิเศษช่วง market regime shift และ volatility spike

## การนำไปใช้กับ crypto bot
- แทน equity price ด้วย BTC/USDT OHLCV
- แทน financial news ด้วย crypto news feed + Twitter/Reddit sentiment
- Architecture GAN เหมาะกับ crypto เพราะ distribution เปลี่ยนบ่อย
- ใช้เป็น signal layer เพิ่มเติมจาก VP-MACD
