# Kalshi + BTC: Macro Events Forecast Crypto Vol

## สรุปสั้น
Kalshi prediction market probability shifts สำหรับ macro events (Fed rate, CPI, recession) predict BTC volatility ล่วงหน้า 5 วัน Fed signal แข็งแกร่ง in-sample (t=3.63) ส่วน Recession risk ดีกว่า out-of-sample (MSFE 0.979)

## สองช่องทางสำคัญ

### BTC: Monetary Policy Channel
```
Fed-dovish signal → BTC vol
  In-sample:  coeff +0.639, t=3.63, p<0.001
  Out-of-sample: MSFE = 1.009 (อ่อนกว่า)
  
Recession Risk → BTC vol
  In-sample:  coeff -1.536, t=-2.76
  Out-of-sample: MSFE = 0.979* (ดีกว่า — more stable)
```

### Altcoins: CPI Channel

| Asset | Coeff | t-stat | OOS MSFE |
|-------|-------|--------|----------|
| Ethereum | -1.209 | -2.12 | **0.959*** |
| Solana | -0.850 | -2.55 | **0.983*** |
| Chainlink | -1.262 | -3.39 | 0.992 |

CPI probability shift → altcoin vol ลดลง (uncertainty resolution = vol compression)

## Implementation: Macro Event Calendar Filter

```python
import requests
from datetime import datetime, timedelta

MACRO_EVENTS = {
    "fed_decision":   {"asset": "BTC", "signal": "fed_dovish",    "lookforward": 5},
    "cpi_release":    {"asset": "ETH", "signal": "cpi_repricing", "lookforward": 5},
    "nonfarm_payroll": {"asset": "BTC", "signal": "recession_risk", "lookforward": 5},
}

def get_kalshi_probability(contract_ticker):
    """ดึง probability จาก Kalshi API"""
    # https://kalshi.com/markets/KXFED / KXCPI / KXRECSSNBER
    resp = requests.get(f"https://trading-api.kalshi.com/v1/markets/{contract_ticker}")
    return resp.json()["market"]["last_price"]

def compute_vol_adjustment(asset, date_today):
    """
    คำนวณ predicted_vol_multiplier จาก macro signals
    ใช้ปรับ position size: weight = baseline_vol / predicted_vol
    """
    if asset == "BTC":
        recession_prob = get_kalshi_probability("KXRECSSNBER")
        # recession risk สูง → BTC vol สูง → ลด position
        vol_multiplier = 1.0 + abs(-1.536 * recession_prob)

    elif asset in ["ETH", "SOL"]:
        cpi_change = get_kalshi_daily_change("KXCPI")
        # CPI resolution (high absolute change) → vol ลด
        vol_multiplier = max(0.5, 1.0 - 0.16 * abs(cpi_change))

    return vol_multiplier

def macro_adjusted_position(base_size_usd, asset, baseline_vol=0.02):
    vol_mult = compute_vol_adjustment(asset, datetime.today())
    predicted_vol = baseline_vol * vol_mult
    # vol targeting: ลด size ถ้า predicted vol สูง
    return base_size_usd * (baseline_vol / predicted_vol)
```

## ตัวเลขสำหรับ position sizing
- 90th percentile Fed signal → ลด BTC allocation ~7%
- CPI uncertainty resolution → altcoin vol ลด ~16%

## Event Calendar
- Fed decision: ทุก 6 สัปดาห์ ตรวจ KXFED 2-3 วันก่อน
- CPI release: ทุกเดือน ตรวจ KXCPI 1 วันก่อน
- NFP: ทุกเดือน ตรวจ KXRECSSNBER

## ข้อควรระวัง
- Fed channel ทำงานดีในช่วง active rate repricing (2024-2025) ไม่ใช่ทุก regime
- Recession risk signal stable กว่า → ใช้เป็น default filter
- Kalshi API ฟรีสำหรับ historical ต้อง paid สำหรับ real-time
- Lag signal 1 วัน (ใช้ data เมื่อวาน predict วันนี้ — no lookahead)
