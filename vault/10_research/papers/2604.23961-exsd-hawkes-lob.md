# ExsdHawkes: State-Dependent Order Arrival Model

## สรุปสั้น
Model order arrival intensity ที่ขึ้นกับสถานะ bid-ask spread ของ LOB ณ ขณะนั้น Key insight: ตอน spread กว้าง (disequilibrium) system เข้าสู่ super-criticality (branching ratio > 1) ทำให้เกิด volatility spike cascade

## Core Formula

### State-dependent intensity
```
λ̃ₑₓ(t) = φₑ(X(t), x) · [νₑ + Σ ∫ kₑ'ₑ(t-s, x') dÑₑ'ₓ'(s)]
```
- φₑ ∈ {0,1} = admissibility mask (กัน physically impossible transitions)
- νₑ = baseline intensity
- kₑ'ₑ = excitation kernel (exponential decay)

### Recursive estimation (O(N) efficient)
```
Rₑ'ₓₑ(n) = Rₑ'ₓₑ(n-1) · exp(-βₑ'ₓₑ · Δt) + 𝟙{Eₙ₋₁=e', Xₙ₋₁=x} · αₑ'ₓₑ
```

### Local branching ratio (Super-criticality detector)
```
nₑₓ = Σₑ' αₑ'ₓₑ / βₑ'ₓₑ
```
- nₑₓ > 1 = explosive clustering = volatility spike coming
- Paper พบ nₑₓ ≈ 2.67 ตอน spread ≥ 2 ticks

## Dataset และผล
- Data: Mitsubishi UFJ (8306) tick data, Oct-Dec 2020, 100-microsecond resolution
- ExsdHawkes reproduce volatility signature plot (upward slope) ได้ — SD-Hawkes ธรรมดาทำไม่ได้
- Residual near-zero autocorrelation ใน 28 state-event pairs

## Critical Finding
Marketable Limit Orders (MLOs) trigger spread-widening → local super-criticality → intense price fluctuations ในไม่กี่วินาที

## Python Implementation (Simplified)

```python
import numpy as np
from collections import defaultdict

class ExsdHawkesLOB:
    def __init__(self, event_types=14, max_spread_state=5):
        self.E = event_types
        self.X = max_spread_state
        self.nu = np.ones(event_types) * 0.1
        self.alpha = np.random.uniform(0, 0.05, (event_types, max_spread_state, event_types))
        self.beta = np.ones((event_types, max_spread_state, event_types)) * 2.0
        self.phi_mask = np.ones((event_types, max_spread_state, max_spread_state))

    def branching_ratio(self, x_state):
        """nₑₓ > 1 → super-critical → danger zone"""
        n_ex = np.zeros(self.E)
        for e in range(self.E):
            for ep in range(self.E):
                if self.beta[ep, x_state, e] > 0:
                    n_ex[e] += self.alpha[ep, x_state, e] / self.beta[ep, x_state, e]
        return n_ex

    def is_supercritical(self, x_state, threshold=1.0):
        return np.max(self.branching_ratio(x_state)) > threshold
```
