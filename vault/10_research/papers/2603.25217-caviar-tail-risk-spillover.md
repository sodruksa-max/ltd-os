# CAViaR-SE: Tail Risk ที่มาจากสินทรัพย์อื่น

## สรุปสั้น
แยก VaR ออกเป็น 2 ส่วน: risk ของตัวเอง + risk ที่ลามมาจากสินทรัพย์อื่น (spillover) spillover อธิบาย 20-80% ของ tail risk รวม ใช้เป็น early warning ว่า contagion กำลังมา

## Core Formula

```
VaRᵢ,ₜ = qᵢ,ₜᵖ + qᵢ,ₜˢ
```

### Proper-risk component (ตัวเอง)
```
qᵢ,ₜᵖ = ω + β₁·qᵢ,ₜ₋₁ᵖ + β₂·|rₜ₋₁|
```

### Spillover component (จากสินทรัพย์อื่น)
```
qᵢ,ₜˢ = φ₁·qᵢ,ₜ₋₁ˢ + φ₂·𝐲ₜ₋₁·𝐰ᵢ
```
- 𝐲ₜ₋₁ = VaR vector ของ asset ทั้งหมด
- 𝐰ᵢ = spillover weights (จาก recursive partial correlation)
- φ₂ > 0 → spillover amplify tail risk

## ผล
- Spillover อธิบาย **อย่างน้อย 20%** ของ total tail risk (บาง asset ถึง 80%)
- φ̂₂ > 0 consistently across 29 Dow Jones stocks
- CAViaR-SE ชนะ baseline CAViaR ใน OOS Model Confidence Set

## Python Implementation

```python
import numpy as np
from scipy.optimize import minimize

class CAViaRSE:
    def __init__(self, alpha=0.05):
        self.alpha = alpha  # tail probability (VaR level)
        self.params = None  # [omega, beta1, beta2, phi1, phi2, weights...]

    def fit(self, returns, spillover_assets_returns):
        """
        returns: T-array ของ asset ที่สนใจ
        spillover_assets_returns: T×K array ของ influential assets
        """
        def objective(params):
            var_series = self._compute_var(returns, spillover_assets_returns, params)
            # Quantile regression loss
            residuals = returns - var_series
            loss = np.where(residuals < 0,
                           (self.alpha - 1) * residuals,
                           self.alpha * residuals)
            return np.mean(loss)

        self.params = minimize(objective, x0=np.zeros(5 + spillover_assets_returns.shape[1])).x

    def _compute_var(self, returns, spill_returns, params):
        omega, beta1, beta2, phi1, phi2 = params[:5]
        weights = params[5:]
        T = len(returns)
        q_proper = np.zeros(T)
        q_spill = np.zeros(T)

        for t in range(1, T):
            q_proper[t] = omega + beta1 * q_proper[t-1] + beta2 * abs(returns[t-1])
            spillover_input = spill_returns[t-1] @ weights
            q_spill[t] = phi1 * q_spill[t-1] + phi2 * spillover_input

        return q_proper + q_spill

    def spillover_ratio(self, returns, spill_returns):
        """สัดส่วน VaR ที่มาจาก spillover"""
        var = self._compute_var(returns, spill_returns, self.params)
        q_proper = ...  # คำนวณ proper component อย่างเดียว
        return 1 - (q_proper / var)

    def alert(self, current_spill_ratio, threshold=0.25):
        """แจ้งเตือนถ้า spillover เกิน 25% ของ total VaR"""
        return current_spill_ratio > threshold
```
