# QRAFTI: Agentic Quant Research Framework

## สรุปสั้น
Framework ที่ให้ AI 3 ตัว (Research Analyst + Risk Manager + Quant Developer) ทำงานร่วมกันด้วย MCP โดยมี reflection-based planning ก่อน execute ทำให้ multi-step accuracy ขึ้นจาก 74% → 99%

## Architecture

```
User Request
    ↓
Factor Research Agent (Research Analyst)
  - interpret + decompose task
  - เลือก tools
  - วาง pathway
    ↓
Reflection Phase (3 steps):
  1. Draft sequential plan
  2. Critique implementability
  3. Refine before execute
    ↓
Code Writing Agent (Quant Developer)
  - เขียน + รัน Python เมื่อ tool ไม่พอ
    ↓
Reporting Agent (Risk Manager)
  - diagnostics + narrative report
```

## Tool-Calling System
Tools expose เป็น callable functions:
- Primitive ops (math operators)
- Cross-sectional transforms (winsorize, quantile bin, standardize)
- Time-series transforms (rolling, lag, resample)
- Portfolio construction (characteristic → weights → returns)
- Standardized reporting templates

## Performance Results

| Task Type | Tools only | Tools + Reflection | Code only |
|-----------|-----------|-------------------|-----------|
| 1-step | 96.75% | — | 72.13% |
| 2-step | 100% | — | 59.10% |
| Multi-step | 74.35% | **98.58%** | — |

**Reflection เพิ่ม multi-step accuracy จาก 74% → 99%** — สำคัญมากสำหรับ trading pipeline

## Key Insight
**Reflection-based planning คือ key differentiator** — ไม่ใช่แค่จำนวน agents แต่คือการ critique plan ก่อน execute ที่ทำให้ระบบแม่น
