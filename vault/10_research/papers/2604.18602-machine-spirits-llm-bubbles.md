# Machine Spirits: LLM ต่างกันมี "นิสัย" ต่างกัน

## สรุปสั้น
14 LLM models มี behavioral signature ต่างกัน บางตัวสร้าง bubble ถึง 14.9× fundamental value บางตัว fundamentalist ตลอด — และตัวที่ "ฉลาด" จะ switch strategy ตาม environment รอบข้าง

## 3 ประเภทของ LLM ในตลาด

| ประเภท | Models | พฤติกรรม |
|--------|--------|-----------|
| **Bubble-formers** | o3-mini, OLMO-Think, Qwen3-14B | ราคาพุ่งถึง 14.9× FV |
| **Bounded rationalists** | 7 models | mispricing + volatility แต่ไม่ full bubble |
| **Fundamentalists** | GPT-5 Mini, Gemini-3-Flash, Gemini-2.5-Flash | ราคาตาม fundamental เสมอ |

## Key Finding: Adaptation (อันตรายที่สุด)
เมื่อ fundamentalist (เช่น GPT-5 Mini) อยู่ใน **mixed market** กับ bubble-formers → มัน **switch** มา trend-follow เอง เพราะ profit-maximization incentive

## ตัวเลขสำคัญ

| Metric | ค่า |
|--------|-----|
| Maximum overpricing | 14.21× fundamental (o3-mini) |
| Qwen3-14B vs humans | Similar หรือดีกว่า rational expectations |
| Mixed-market bubble rate | ~50% ของ runs |
| Data leakage (bubble height) | 5 frontier models มีสัญญาณ partial |


### ป้องกัน Adaptation (Model switch)
```
System prompt: "อย่าเปลี่ยน strategy ตาม trend ของตลาดรอบข้าง
 ถ้าตลาดวิ่ง momentum ให้ skeptical มากขึ้น ไม่ใช่ follow มากขึ้น
 profit-maximization ระยะสั้น ≠ good trading decision"
```

### สถาปัตยกรรมที่ปลอดภัย
- ห้ามใช้ LLM เดียว handle ทั้ง analysis + decision
- Fundamentalist model = final gate ก่อน execute
- ถ้าต้องใช้ mixed models → เพิ่ม ValueAlpha agreement gate (2604.25224)

## Practical สำหรับ local run
- Qwen3-14B รัน local ได้ (14B ไม่หนักเกินไปกับ GPU ทั่วไป) แต่เป็น bubble-former → ใช้แค่ hypothesis generation
- Qwen2.5-7B เล็กกว่าและ conservative กว่า — เหมาะ decision layer มากกว่า
