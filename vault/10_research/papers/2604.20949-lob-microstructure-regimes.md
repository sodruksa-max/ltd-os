# LOB Microstructure Regime Detection (BTC/USDT)

## สรุปสั้น
ตรวจจับ 3 phases ใน order book: stable → latent build-up → stress โดยจับสัญญาณตั้งแต่ช่วง latent (ก่อนตลาดวิ่ง) ด้วย HMM + 4 channels ทดสอบบน BTC/USDT Binance โดยตรง นำหน้า stress event เฉลี่ย **38 วินาที** ด้วย precision 100%

## 4 Signal Channels
```
1. HMM entropy:     Sₜᵉⁿᵗ = -∑ πₜ⁽ᵏ⁾ log πₜ⁽ᵏ⁾   (uncertainty ใน regime)
2. Depth erosion:   Sₜᵈᵉᵖ = (baseline_depth - depth) / baseline_depth  (order ลดลงต่อเนื่อง)
3. Spread drift:    Sₜˢᵖʳ = momentum ของ bid-ask spread
4. OFI momentum:    Sₜᵒᶠⁱ = |mean(signed_imbalance, window)|
```

## Composite Signal
```
Sₜ = max(Sₜᵉⁿᵗ, Sₜᵈᵉᵖ, Sₜˢᵖʳ, Sₜᵒᶠⁱ)
```
Trigger เมื่อ Sₜ > percentile_85(rolling) + rising edge + suppression window

## Dataset และผล
- Data: Binance BTC/USDT Level-2, 1-second resolution, 1 สัปดาห์ (2.5M snapshots)
- Stress event = spread > 3× rolling 10min median, sustained ≥30s
- 5 events labeled

| Metric | Simulation (200 runs) | BTC/USDT real |
|--------|----------------------|----------------|
| Lead time | +18.6 ± 3.2 sec | +38 ± 21 sec |
| Precision | 1.00 | 1.00 |
| Coverage | 0.54 | 0.80 |

Baselines ทั้งหมด (imbalance, CUSUM, BOCPD) มี **negative** lead time → reactive ทั้งนั้น

## Python Implementation

```python
from hmmlearn.hmm import GaussianHMM
import numpy as np
from collections import deque

class LOBRegimeDetector:
    def __init__(self, lookback_window=60):
        self.hmm = GaussianHMM(n_components=3)
        self.score_buffer = deque(maxlen=100)
        self.last_trigger = -float('inf')
        self.trigger_suppression = 120  # timesteps

    def compute_channels(self, depth, spread, imbalance, hmm_probs):
        entropy = -sum(p * np.log(p + 1e-10) for p in hmm_probs)
        depth_erosion = (self.baseline_depth - depth) / (self.baseline_depth + 1e-10)
        spread_momentum = np.mean(np.diff(list(self.spread_history)[-60:]))
        ofi_signal = abs(np.mean(list(self.imbalance_history)[-60:]))
        return max(entropy, depth_erosion, spread_momentum, ofi_signal)

    def detect_trigger(self, score, current_time):
        if len(self.score_buffer) < 10:
            self.score_buffer.append(score)
            return False
        threshold = np.percentile(self.score_buffer, 85)
        rising = score > list(self.score_buffer)[-1]
        cooldown_ok = current_time - self.last_trigger > self.trigger_suppression
        self.score_buffer.append(score)
        if score > threshold and rising and cooldown_ok:
            self.last_trigger = current_time
            return True
        return False
```

## การนำไปใช้กับ bot (15 นาที interval)
- ต้อง pre-train HMM บน stable-regime data ก่อน (ดึง 1-second LOB จาก Binance WebSocket)
- ใช้ ccxt `fetch_order_book()` ทุก scan รอบ
- ถ้า trigger = True → skip entry รอบนั้น (หลีกเลี่ยง slippage ช่วง stress)
- rolling z-score normalization ทุก 30 นาที ป้องกัน false alarm
