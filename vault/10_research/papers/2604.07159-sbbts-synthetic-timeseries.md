# SBBTS: Synthetic BTC Price Data สำหรับ Backtest

## สรุปสั้น
Framework สร้าง synthetic financial time series ที่ยึด distribution จริงของตลาดด้วย optimal transport theory ต่างจาก GAN ตรงที่ calibrate ทั้ง drift และ volatility พร้อมกัน ใช้ backtest bot ใน stress scenario ก่อนต่อ API จริง

## Core Math

### Objective
```
min J(ℙ) = 𝔼ℙ[∫₀ᵀ ‖αₜ‖² + β‖σₜ - Iₐ‖² dt]
```
- α = drift
- σ = volatility
- β = trade-off parameter (β ใหญ่ = เน้น volatility calibration)
- Subject to: marginal distribution constraints ที่ t₀, t₁, ..., tₙ

### Solution components
```
Transport map:  𝒴ₜ(x) = x - (1/β)∇ᵧ log hₜ(x)
Drift:          αₜ* = ∇ᵧ log hₜ(𝒴ₜ(Xₜ))
Volatility:     σₜ* = D²ᵧΦₜ(𝒴ₜ(Xₜ))
```

## ข้อดีเหนือ GAN
- Recover stochastic volatility parameters (ξ, ρ) ที่ GAN ทำไม่ได้
- Preserve autocorrelation + cross-sectional correlation ระหว่าง assets
- Mathematical guarantee ว่า marginal distributions match
