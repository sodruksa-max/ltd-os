# MPC Trade Execution: ลด Slippage ด้วย Optimal Control

## สรุปสั้น
ใช้ Model Predictive Control (dynamic programming) เพื่อ optimize วิธี execute order ผล: ลด schedule shortfall 40-50% เทียบกับ naive TWAP/VWAP ทดสอบบน NASDAQ Level 3 data 6 เดือน solve เร็ว ~1ms ต่อ step

## Core Objective Function

```
minimize: (c_t ∘ π_t)′u_t + γ(q_t + π_t′u_t − s_{t+1})² + ξ_t(Q − q_t − π_t′u_t)
```

- `c_t` = trading cost per price level
- `π_t` = fill probability vector
- `u_t` = order quantities (control variable)
- `γ` = schedule adherence weight
- `s_{t+1}` = schedule target (TWAP/VWAP/AC)
- `ξ_t` = future cost estimate (rollout)
- `Q` = total quantity to execute

## Constraints

```
u_t ≥ 0                           (non-negative orders)
u_t ≤ κ                            (max 50% per order)
q_t + π_t′u_t ≤ s_{t+1} + ρ_upper (don't overshoot schedule)
q_t + 1′u_t^m ≥ s_{t+1} − ρ_lower  (market order if too far behind)
u_t′Σ_t u_t ≤ β                    (variance/risk bound)
```

## ผล
- **40-50% reduction** ใน schedule shortfall vs spread-crossing baseline
- Solve time: ~1ms (Clarabel interior-point solver) — real-time capable
- Oracle (ใส่ price forecast เข้า ξ_t): +160% performance improvement

```python
class MPCTradeExecutor:
    def __init__(self, gamma=1.0, beta=0.01):
        self.gamma = gamma   # schedule tightness
        self.beta = beta     # variance bound
        
    def execute_step(self, state, schedule_target, order_book):
        """Solve QP at each time step"""
        costs = self.cost_model(order_book)           # c_t
        fill_probs = self.fill_prob_model(order_book) # π_t
        cov = self.fill_cov_model(order_book)         # Σ_t
        future_cost = self.rollout_cost(state)        # ξ_t
        
        # QP: minimize objective subject to constraints
        # solve with Clarabel (~1ms)
        u_optimal = qp_solve(costs, fill_probs, cov,
                             state.executed_qty, schedule_target,
                             gamma=self.gamma, beta=self.beta)
        return u_optimal
    
    def fast_cancel(self, orders, price_moved):
        """Cancel limit orders ถ้าตลาดวิ่งผิดทิศ"""
        if price_moved:
            return [o for o in orders if o.type == 'market']
        return orders
```

```python
# Simplified slippage control
def smart_order(quantity, price, order_book, max_slippage=0.001):
    """Split order ถ้าขนาดใหญ่เกินกว่า top-of-book"""
    top_bid_qty = order_book['bids'][0][1]
    if quantity > top_bid_qty * 0.3:
        # แบ่งเป็น 3 child orders
        return [quantity/3, quantity/3, quantity/3]
    return [quantity]
```
