# VP-MACD

## สรุปสั้น
อัปเกรด MACD ธรรมดาด้วยการแทน closing price ด้วย adjusted price ที่รวม volume + volatility + intraday conviction เข้าไป ผลคือ signal น้อยลงแต่แม่นขึ้น และมีพารามิเตอร์ λ ให้จูนจังหวะ entry ให้เร็วขึ้น

## Core Formula

### Adjusted Price (แทน close ปกติ)
```
P*_t = Σ(P_i · Volume_i · σ_i · r_i) / Σ(Volume_i)
```
- window N วันย้อนหลัง
- σ_i = volatility component
- r_i = intraday conviction

### Volatility Component
```
σ_i = STD(High - Low) / Close
```
ความผันผวน intraday normalize ด้วย close

### Intraday Conviction
```
r_i = |Close - Open| / (High - Low)
```
วัดว่า candle มี body ใหญ่แค่ไหนเทียบกับ wick รวม (body ใหญ่ = conviction สูง)

### VP-MACD และ Signal Line
```
VP-MACD_t = EMA_12(P*_t) - EMA_26(P*_t)
Signal_t   = EMA_9(VP-MACD_t)
```
ใช้ EMA 12/26/9 เหมือน MACD ปกติ แต่คำนวณบน P* แทน close

## Sensitivity Parameter λ
- Range: λ ∈ (0.8, 1.0)
- λ < 1 → ลด threshold ฝั่ง buy ทำให้ entry เร็วขึ้นก่อน crossover จริง

## Entry/Exit Rules
```
Buy:  VP-MACD_(t-1) ≤ λ·Signal_(t-1)  AND  VP-MACD_t > λ·Signal_t
Sell: VP-MACD_(t-1) ≥ Signal_(t-1)     AND  VP-MACD_t < Signal_t
```
**asymmetric**: buy ใช้ λ-adjusted, sell ใช้ standard — ให้ exit เร็ว แต่ entry รอ confirm

## Implementation Steps
1. คำนวณ σ_i และ r_i สำหรับแต่ละแท่งเทียน
2. คำนวณ P* (rolling volume-weighted ที่รวม σ และ r)
3. สร้าง EMA 12/26/9 บน P*
4. เปรียบเทียบ VP-MACD กับ λ·Signal สำหรับ buy, Signal สำหรับ sell
5. Execute ที่ open ของแท่งถัดไป (ไม่ใช้ close เพื่อหลีกเลี่ยง lookahead bias)

## Dataset และผลทดสอบ
- Train: 2018–2022 | Test: 2023–Feb 2026
- Index: S&P 500, Nasdaq-100, DJIA
- ผล: ดีกว่า MACD ธรรมดาใน profitability, Sharpe, และ downside risk control

## Python snippet
```python
import numpy as np
import pandas as pd

def vp_macd(df, N=20, fast=12, slow=26, signal=9, lam=0.9):
    sigma = df['high'].sub(df['low']).rolling(N).std() / df['close']
    r = (df['close'] - df['open']).abs() / (df['high'] - df['low']).replace(0, np.nan)
    p_star = (df['close'] * df['volume'] * sigma * r).rolling(N).sum() / df['volume'].rolling(N).sum()
    ema_fast   = p_star.ewm(span=fast, adjust=False).mean()
    ema_slow   = p_star.ewm(span=slow, adjust=False).mean()
    vp_macd_line = ema_fast - ema_slow
    signal_line  = vp_macd_line.ewm(span=signal, adjust=False).mean()
    buy  = (vp_macd_line.shift(1) <= lam * signal_line.shift(1)) & (vp_macd_line > lam * signal_line)
    sell = (vp_macd_line.shift(1) >= signal_line.shift(1)) & (vp_macd_line < signal_line)
    return vp_macd_line, signal_line, buy, sell
```
