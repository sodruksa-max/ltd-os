# ValueAlpha: Stress-Test LLM ก่อนเทรด

## สรุปสั้น
ทดสอบ rationale ของ LLM 4 ตัว (Claude, GPT, Gemini, Qwen) โดยใช้ judge panel 3 ตัวและ weighted Cohen's kappa (κ) วัด agreement ถ้า agreement ต่ำ = ห้าม publish / ห้าม execute เป็น framework governance ก่อนที่คำสั่ง trade จะออกจริง

## Agreement Gate (หัวใจของระบบ)
```
κ̄w ≥ 0.4  → Publish / Execute allowed
0.2 ≤ κ̄w < 0.4  → Methodology finding only — no headline claim
κ̄w < 0.2  → HALT — no ranking, post-mortem only
```

## ผลสำคัญ 5 ข้อ

| Result | สิ่งที่ค้นพบ |
|--------|-------------|
| 1 | dimension constraint_awareness fail (κ̄w=0.2022) — แม้ aggregate pass |
| 2 | rank 2-4 collapse เป็น tie-class — มีแค่ rank-1 (Claude) ที่ stable จริง |
| 3 | single judge ให้ ρ=0.2–0.8 variance — ต้องใช้ ensemble เสมอ |
| 4 | rationale สั้น-ถูกโดนหัก 2.81 คะแนน vs rationale ยาว — instrument มี style bias |
| 5 | เปลี่ยน anchor phrasing → ranking order พลิก — ต้องใช้ตัวเลขเสมอ |
